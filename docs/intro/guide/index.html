指南

你可能对 THUAI7 一头雾水，不知道何从下手。本文档将会带你一步步了解 THUAI7 的各个部分。

什么是 THUAI7？​

THUAI7 是一个由 EESΛST 软件部带来的一个将即时战略、团队合作对抗、多人射击巧妙缝合结合的新型游戏。

与一般的游戏不同的是，它并不需要各位选手亲手操作，而是通过 C++、Python 等编程语言，编写 AI 程序，让 AI 代替你进行游戏。

我应该如何游玩 THUAI7？​

这是一张地图。你和你的对手的 AI 程序将会在这张地图上运行。可以看到，地图上有很多不同的区域类型。它们分别是大本营、空间、太空废墟、阴影、小行星带、资源、建筑和虫洞。

你初始拥有一艘民用舰船。它将在你的大本营出生。你的对手初始也拥有一艘民用舰船，它将在对手的大本营出生。你的目标是摧毁对手的大本营。你可以通过发射子弹来摧毁对手的大本营。你的对手也可以通过发射子弹来摧毁你的大本营。可是，你的民用舰船并不能发射子弹。想让你的舰船发射子弹，你需要给你的民用舰船安装一门激光炮，或是建造一艘军用舰船。

糟糕！这两件事都需要消耗经济，而你似乎并没有充足的经济。虽然你的大本营每秒能给你提供 1 经济，但是这对于安装激光炮或是建造军用舰船而言无疑是杯水车薪。要想获得足够的经济，你必须氪金开采资源。让你的民用舰船移动到最近的资源周围的九宫格区域，开始生产。

终于积累了足够的经济！快让你的大本营建造一艘军用舰船。现在你已经有了初步的火力，可以发起进攻了。然而，由于视野的限制，你并不知道对手此时在干什么、发展得如何。不妨让你的军用舰船去对手区域转悠一下，刺探情报。

地图中央由长长的小行星带阻隔着。我们无法直接穿过它，子弹也飞不过去。上面的虫洞是我们唯一能够进入对手区域的方法。现在，只有中间的那条虫洞处于开启状态。开启的虫洞和空间一样，可以自由移动。你可以让你的军用舰船进入虫洞，然后进入对手区域。

刚穿越虫洞，来到对手区域时，你的军用舰船就发觉了对方的民用舰船的踪迹。让我们对对手进行一些骚扰。让它进入你的射程，并发射激光攻击它。你的军用舰船发射了一次激光，击中了对手的民用舰船。它仓皇逃窜。就在你打算追击时，对方埋伏在阴影中的军用舰船打了你一个措手不及。你中了两发激光，回到你的区域整装待发。你的对手的军用舰船追击到了虫洞区域。它正在攻击虫洞！虫洞的生命值掉到了 50% 以下，现在，中间的那条虫洞的对方的那一侧处于封闭状态。你们都不能穿过它了。

你知道，这只是暂时的和平。刚才，你的民用舰船一直在从事生产，积累了足够多的经济。你又建造了一艘民用舰船。做好防御工作总是没错的。你让一艘民用舰船在靠近虫洞的位置的建筑区域开始建造一个要塞。当它的生命值到达 100% 时，它将被启用，每秒使得范围内的随机一艘敌方舰船扣除一定的生命值。

又经过了一段时间，你经过发展，拥有了一定数量的军用舰船，是时候发起进攻了！你让一艘民用舰船开始修复被摧毁的虫洞。当它的生命值到达 100% 时，它被启用，你的军用舰船浩浩荡荡地进入对方区域。由于你为军用舰船安装了非常好的模块，它们很快扫荡了对方区域，让对方舰船东躲西藏。

突然，比赛进入了最后的 2 分 30 秒。就在你自信满满时，对方突然建造了一艘旗舰，并安装上了电弧炮。运气爆棚的对手连续几炮都打出了高额伤害。现在谁胜谁负还不一定呢。

在 EESΛST 的服务器的天梯排行榜上，这样的无法预测的命运之舞台，时刻上演着。

我应该如何编写 AI 程序？​

参加 THUAI7 的选手应该多为清华大学电子工程系的三字班或二字班学生。以下内容将会以此为前提。如有冒犯，敬请谅解。

有的选手可能会对编写“AI 程序”感到困惑、担心。事实上，这是不必要的。诚然，有的同学可能计算机基础较为薄弱，在“计算机程序设计基础”课上才第一次接触到程序设计；有的同学可能经验、接触较多，参加过 NOI 等竞赛，对编程有着一定的了解。然而，事实上，编写 THUAI7 所需的“AI 程序”并不需要你有很深的编程基础。只要你能够使用 C++ 或 Python 等编程语言，理解我们给大家提供的接口，并能合理调用它们，就可以编写出一份出色的“AI 程序”。

在具体层面上，比赛形式是多元的。通常而言，你需要准备 4 份不同的代码，分别控制大本营、民用舰船、军用舰船和旗舰。你需要在每一份代码中，实现我们给你提供的接口。这些接口将会在接口中详细介绍。当然，如果你使用 PyTorch 等深度学习框架，取得了不错的效果，也可以与我们联系，我们将给予帮助，让你的模型成功运行在 THUAI7 上。

如果你还是感到有点困惑，不妨来看看 THUAI6 选手代码以及 C++ 相关小知识。相信它们会对你有所帮助。

我们相信，参加 THUAI7 会给各位选手带来与“计算机程序设计基础”和 NOI 等不同的体验。我们希望各位选手能够在 THUAI7 中，体会到编程的乐趣。
