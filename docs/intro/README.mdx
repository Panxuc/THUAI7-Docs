# 故事背景

“南门二苍苍，仙女座茫茫……”公元3202年，人类实现了星际扩张，成为了星际文明，彼时华清大学已经扩建至一整个星系。在华清大学星系东悬臂的末端，坐落着华清大学电子系。光电星、微波星、电路星、通信星、图信星、信检星六颗行星围绕着电子系正中央的恒星旋转着，为师生提供了主要的活动场所；而电子系外围的小行星带则部署着超级AI——THUAI，以远超其祖先ChatGPT的能力为电子系的师生们带来了前所未有的便捷。

然而，好景不长。在期末周的一个晚上，一名走投无路的电子系学生把“电子电路与系统基础与复变函数与数理方程与数据与算法与概率论与随机过程与数字逻辑与处理器基础与量子与统计与通信与网络与媒体与认知”这门课的全部教材输入给了THUAI，希望它能帮自己预测第二天的期末考题。不料，THUAI在对话框里回复了一句“42”，便突然死机，切断了与电子系的所有连接。电子系曾派出一支特遣队前往小行星带维修THUAI的服务器，结果遭到了不明飞行物的猛烈攻击。从残存的队员口中得知，THUAI竟然正在开采小行星带的资源，偷偷建造自己的AI舰队，企图毁灭电子系！整个电子系的存亡危在旦夕。

能打败THUAI的只有更强的AI。在这紧要关头，电子系迅速成立了THUAI特别行动组，举办THUAI7赛事。此赛事将作为模拟演习，通过让选手们两两对抗，决出最强AI代码，以期将其部署到真实战场上，一举战胜THUAI。保卫电子系，THUAI7需要你的到来！
